1、该例程为I2C例程。

2、使用说明
   (1)工程文件路径：例程目录\I2C+USART\软件I2C\MDK-ARM\Project.uvproj。
   (2)请使用MDK 4.0以上版本打开,MDK版本过低会导致无法识别工程。
   (3)下载调试工具为ULINK。
   (4)请将串口线(直连串口线)插在板子COM1口上,并打开超级终端或串口助手，配置波特率115200，8位，一个停止位，无校验位。
   (5)HEX文件下载到板子后,LED灯闪烁,使用超级终端或串口调试助手可以看到调试信息,表明例程运行正确。
	   	
3、注意事项
   请务必在下载、调试、运行过程中,保持板子上电、ULINK连接并插在电脑上。
